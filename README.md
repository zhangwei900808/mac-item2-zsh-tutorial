# mac-iterm2-zsh-tutorial

## 1、什么是Iterm2？
用Iterm2自己的话说就是：iTerm2是终端的替代品，也是iTerm的继承者。 它适用于使用macOS 10.10或更高版本的Mac。 iTerm2将终端带入了您从未想过的功能的现代时代。

## 常用快捷键
⌘+N、⌘+T、⌘+↔、⌘+F、⌘+W 这几个快捷键就不解释了。
⌘+D 垂直分屏、⌘+⇧+D 水平分屏、⌘+⌥+↑↓↔ 切换分屏。
⌘+↩全屏、⌘ + R 清屏。
⌃+A/E 行首/尾、⌃+R 查询历史命令。
⌥+↔ 左右跳过单词。
记住这些就差不多了，脑容量有限，应能省则省。
